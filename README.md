# FinanceTracker


## 🗺️ Roadmap: Building a Finance Tracker in C++ & Qt

### How to run:

__*Linux/Ubuntu*__

```
mkdir -p build
cd ~/Desktop/FinanceTracker/build && cmake .. && cmake --build . -j$(nproc) && ./FinanceTracker
```

__*Windows*__

```
mkdir build && cd build
cmake ..
cmake --build .
./FinanceTracker

```
